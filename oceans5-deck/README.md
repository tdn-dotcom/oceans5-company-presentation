## Tech Stack

- **[reveal.js 5.1](https://revealjs.com/)** — HTML presentation framework (CDN)
- **[Chart.js 4.4](https://chartjs.org/)** — data visualisation (CDN)
- **Custom CSS** — Oceans5 brand theme (`css/oceans5-theme.css`)
- **Google Fonts** — Inter + Montserrat

No build step required. Open `index.html` directly.
